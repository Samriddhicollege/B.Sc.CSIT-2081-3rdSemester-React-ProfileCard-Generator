## Project Overview

> This project is a web-based Profile Card Generator developed using React for the frontend.
> It allows users to manage professional profile cards, featuring full CRUD (Create, Read, Update, Delete) functionality.
> Users can input details such as name, role, and bio, seeing a real-time preview of their card.
> The system utilizes LocalStorage for data persistence to ensure user profiles are saved locally.
> The main goal is to practice React state management, component architecture, and form handling.
